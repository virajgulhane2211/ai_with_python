# AI with Python Virtual Lab

Static GitHub Pages-ready virtual lab website using HTML, CSS, Bootstrap 5 CDN and Vanilla JavaScript.

## Latest Update
- Added 38 complete basic Python programs, one for each virtual lab.
- Updated the Python Code tab inside every lab.
- Added direct downloadable `.py` files in the `programs/` folder.

## Features
- 38 virtual labs mapped to 6 units
- Search and unit/type filters
- Interactive demos for AI, ML, DL, Python and applications
- Basic Python program for every lab
- Viva questions and quiz for every lab
- Printable lab manual pages
- Local progress tracking with localStorage
- No backend, database, Node.js or paid API

## Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html`, `assets/`, and `programs/`.
3. Go to Repository Settings → Pages.
4. Select branch `main` and folder `/root`.
5. Open the GitHub Pages URL.

## Edit Labs
Open `assets/js/app.js` and modify `rawLabs`, `theory`, `code`, or demo functions.

## Programs Folder
Open `programs/README.md` to see all 38 Python program files.
